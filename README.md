# Simple-Babelweb
A very simple Babel Web for babeld v1.8
* Just clone it to your favorite httpd directory.

## index.php
The php version for gateways
### Requirements
* php
* netcat-openbsd (ipv6) 

## babel.html
The haserl version for slim devices without php
### Requirements
* haserl
* netcat with ipv6 support
